# Local setup

Step 1: Download or clone the source code from GitHub to the local machine

# Backend

Step 2:  ```mvn clean install```

Step 3:  ```mvn spring-boot:run```

# Frontend

Step 4:  ```npm install```

Step 5:  ```npm start```

# From the browser call the endpoint http://localhost:3000
